Repository ini bertujuan untuk melakukan pengujian untuk Mini Case FoodFast. Dokumen ini mencangkup pengujian fitur pada aplikasi FoodFast.
Dokumen mencangkup test case, analisis potensi bug, serta saran perbaikan untuk meningkatkan kualitas aplikasi

Isi Repository :
1. Test Case
-Mengandung 6 test case untuk fitur yang ada dalam aplikasi FoodFast. Berisikan tujuan pengujian, langkah-langkah uji coba, data yang diperlukan, dan hasil yang diharapkan.
2. Analisis Potensi Bug
-Mengandung identifikasi 3 potensi bug pada fitur yang berada dalam FoodFast. Berisikan penyebab bug, dampak pada pengguna, dan saran perbaikan.
3. Laporan
- Laporan akhir disusun dalam format Word dan Excel
